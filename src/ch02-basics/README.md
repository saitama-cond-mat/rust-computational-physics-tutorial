# 第2章 数値計算の基礎

> [!IMPORTANT]
> **この章を読む前に**
>
> この章を読むには、以下の章を先に読んでおく必要があります。
>
> - [第1章: Rustと計算物理学](../ch01-introduction/)

第1章では、Rustで計算物理学を学ぶための準備を整えました。本章からはいよいよ、その核心である数値計算の世界に足を踏み入れます。

本章で扱う内容は、これ以降に続くすべての章（第2部「数値計算手法」、第3部「物理シミュレーション」）の基礎となる、非常に重要な土台です。ここでは、コンピュータが数をどのように扱っているかという基本的なテーマから、効率的なベクトル・行列演算を行うためのライブラリまで、Rustで数値計算を行う上で必須となる知識を学びます。

## この章で学ぶこと

- **[浮動小数点演算と誤差](./floating-point.md)**
  コンピュータにおける実数の表現形式である「浮動小数点数」の仕組みと、それに伴って必然的に生じる「丸め誤差」や「桁落ち」といった計算誤差について学びます。これは、数値計算の結果を正しく評価する上で不可欠な知識です。

- **[配列とベクトル演算](./arrays-vectors.md)**
  複数の数値をまとめて扱うための基本的なデータ構造である、Rust標準の配列、スライス、そして`Vec<T>`について解説します。

- **[外部クレートの活用（ndarray入門）](./ndarray.md)**
  PythonにおけるNumPyのように、科学技術計算では多次元配列（ベクトルや行列）を効率的に扱うための専用ライブラリが不可欠です。ここでは、Rustのエコシステムで広く使われている`ndarray`クレートの基本的な使い方を習得します。

- **[高精度演算](./high-precision.md)**
  標準の64ビット浮動小数点数 (`f64`) では精度が不足するような特定の問題に対処するため、`double-double`演算などを用いてより高い精度を実現する手法について紹介します。

本章を終えれば、あなたはRustで数値を正確かつ効率的に扱うための基礎知識とツールを身につけることができます。この土台の上に、第3章以降で様々な数値計算アルゴリズムを構築していきましょう。
